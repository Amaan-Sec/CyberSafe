How they connect

In production, the mobile app pushes RASP detections / scan events to the backend, which the admin console reads via REST. In this prototype the data is mocked on both sides, but the Threat Reports and Audit Trail pages in the admin console show exactly the shape of events the app emits — same threat types, same severity tiers as rasp_service.dart.
